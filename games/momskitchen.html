Mom's Kitchen

Cooking & production management game • Unreal Engine 5.6 • C++

Overview

Mom’s Kitchen is a cooking and shop-management game built in Unreal Engine 5.6 using a C++-first gameplay architecture. The project focuses on modular systems, clean class hierarchies, and scalable production workflows.

Core Systems

  • Inventory & Stackable Items
  • Workstation Base Class Architecture
  • Interactable System Using Interfaces
