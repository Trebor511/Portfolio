Level Renderer

C++ • Gateware • Blender Pipeline • 3D Rendering Systems

Overview

This Level Renderer is a C++ graphics programming project built as an extension of my Rotating Cube renderer. The goal was to evolve the renderer into a fully functional real-time level loader capable of importing custom scene data, reading object transforms, and rendering multiple 3D models efficiently.

The system uses a custom text-based level description format, allowing objects to be positioned, rotated, and loaded dynamically at runtime. Levels were designed in Blender, then exported through a purpose-built script that extracted transform data, enabling accurate scene reconstruction inside the engine.

Core Features

  • Custom Level File Parser: Reads model filenames, positional data, and orientation values from a text-based format to generate the level dynamically at runtime.
  • Model Loading with h2bParser: Integrated the h2bParser to load and manage mesh data from exported .h2b files, improving efficiency and runtime performance.
  • Blender-to-Engine Workflow: Designed levels directly inside Blender, exporting transform data to ensure accurate placement, rotation, and scaling inside the renderer.
  • Object-Oriented Architecture: Implemented a robust system for managing multiple models, improving readability and enabling scalable scene construction.
  • 3D Transformations: Applied translation, rotation, and scaling matrices using Gateware’s math libraries to position objects correctly in world space.

Technical Breakdown

The renderer is built on top of Gateware, which provides low-level access to system hardware, math utilities, and windowing functionality. On top of this foundation, I implemented:

  • Scene generation through iterative parsing of level files
  • Matrix transformation pipelines for world, view, and projection spaces
  • A structured system for storing and rendering multiple mesh instances
  • Optimized batching and model management for real-time performance

This project significantly strengthened my understanding of graphics programming, file I/O, 3D mathematics, engine architecture, and real-time rendering workflows.
