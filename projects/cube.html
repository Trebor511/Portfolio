Rotating Cube

C++ • Computer Graphics • Line Drawing Algorithms • Perspective Projection

Overview

The Rotating Cube is a computer graphics project developed in C++ as part of Full Sail University’s graphics programming curriculum. The assignment began with implementing foundational line-drawing algorithms and gradually expanded into rendering a fully 3D cube, applying rotation transformations, and projecting it into 2D space using perspective mathematics.

The final result is a smooth, real-time animated cube rendered with custom graphics logic. No game engine required, only low-level math, transformation matrices, and manual pixel plotting.

Key Features

  • Line Drawing Algorithms: Implemented foundational rendering logic, including digital differential analyzer (DDA) and Bresenham-style approaches.
  • 3D Cube Geometry: Defined cube vertices, edges, and connectivity manually in 3D space.
  • Rotation Transformations: Applied rotation matrices to animate the cube around multiple axes.
  • Perspective Projection: Used projection math to convert 3D points into 2D screen coordinates for a realistic depth effect.
  • Frame Rendering Loop: Built a continuous update-draw pipeline for smooth animation and real-time performance.
  • CMake Project Structure: Fully configured cross-platform CMake setup for building and compiling the application.

Implementation Notes

This project emphasized understanding the mathematical foundations of 3D graphics. All rotations, projections, and rendering behavior were implemented manually using vector math and transformation matrices—without relying on graphics APIs.

The cube is rendered using per-frame updates that apply incremental rotations, followed by projecting the transformed vertices and drawing the connecting edges. This workflow demonstrates strong fundamentals in graphics programming, linear algebra, coordinate space transformations, and low-level rendering logic.
